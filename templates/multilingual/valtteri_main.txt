# Templates for Valtteri the election bot
#
# This version of the template file specifies templates for the following languages in the multilingual format:
# - Finnish
# - Swedish
# - English
#
# Comments are allowed: any line starting with '#' is ignored.
#
# Headlines are specified with fi-head, sv-head or en-head

# Note that many of the templates below (though not all) use the terms “value”,
# “name” and “place”, which are equivalent to “what_2”, “who_2” and “where_2”

# The templates support also specifying a generic template for pre-determined groups of what_types
# The what_type groups are defined below before the actual templates. The dollar sign is used to differentiate
# lines with group definitions from the actual templates. The brackets in the group names are not strictly necessary,
# but should help differentiate them from ordinary values within templates.
#
# The usage is similar to ordinary what_types, but using a new operator 'in':
#
# fi: {name} sai {value} {unit} [{place, case=ssa}]
# fi: {name} sai {place, case=ssa} {value} {unit}
# en: {name} got {value} {unit} [in {place}]
# | what_type_2 in {base}, what_2 > 0, when_2 = 2017



# ---Group definitions---

${base}: total_votes, percentage_votes
${change}: percentage_votes_change, seats_change, total_votes_change
${vote_change}: percentage_votes_change, total_votes_change
